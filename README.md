# SOC Intern — Tier 1 (Blue Team)

## About
SOC Intern tập trung vào giám sát và phản ứng sự cố mức cơ bản (Tier 1).
Kinh nghiệm chính: phân tích log, xử lý alert, phân loại sự kiện (triage), escalte khi cần thiết, và tuân thủ quy trình Incident Response.

---

## Core Skills

### Monitoring & Detection
- Phân tích log từ SIEM (Splunk / ELK)
- Điều tra alert: brute-force, phishing, malware, suspicious process
- Hiểu mô hình MITRE ATT&CK (mapping basic tactics/techniques)
- Viết query cơ bản (SPL / KQL equivalent)

### Incident Response (Tier 1)
- Alert triage & classification (True/False Positive)
- Thu thập evidence (logs, process tree, network connection)
- Phối hợp escalte lên Tier 2/3
- Viết báo cáo sự cố ngắn gọn, rõ ràng

### Systems & Networking
- Windows (Event Viewer, Sysmon cơ bản)
- Linux log cơ bản (auth.log, syslog)
- TCP/IP, DNS, HTTP, mô hình OSI
- Kiến thức cơ bản về AD, RDP, email security

### Tools
- SIEM: Splunk / Elastic
- EDR cơ bản
- Wireshark (filter cơ bản)
- Nmap (scan căn bản)
- Git & Linux CLI
